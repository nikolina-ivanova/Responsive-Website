What is HTML?

HTML (Hypertext Markup Language) is the code used to build and display a web page and its content. Content can be constructed from several paragraphs, a list of leading characters, or using images and data tables. HTML is not a programming language - it is a markup language that is used to tell your browser how to display the pages we visit. Depending on the intentions of the web designer, the way of rendering can be simple or extremely complex. HTML is a series of elements (called tags) that we use to enclose (or wrap) different parts of the content to make them look and work a certain way.

Main parts of the HTML element

Opening tag

It consists of the name of the element enclosed by opening and closing corner brackets.

Closing tag

Same as the opening tag, but with an additional slash before the item name.

Content

This is the content of the element, in most cases it is text.

Text editors

The user can use any text editor, such as Notepad ++, to create HTML documents. There are also specialized editors such as HTML Assistant 2.0. It is a fairly convenient and easy-to-use functional hyperthers editor that provides the user with a set of buttons for automatically writing HTML elements, but if you use this editor you are very likely not to learn the tags.
