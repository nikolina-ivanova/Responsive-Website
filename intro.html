Getting Started

HTML specifies elements that divide a document into descriptive and functional parts. Each HTML document must be tagged with the <!DOCTYPE html> tag, which indicates the type of document (which version of html we use). Another pair of required tags are <html> and </html>. These tags specify the beginning and end of the HTML document. If desired, we can insert the title tags <head> and </head>, they contain information about the document, which is not displayed with the main text. The <title> Title </title> element is most commonly used in the title part of the document, which marks the title of the HTML document. If this item is used, the title of the document is displayed in the title bar of the browser window. We can also put the <style> and </style>, where we can add a link to another css or javascript file, or enter our css code directly to format the page. The executable part is located between the <body> and </body> tags, which mark the beginning and end of the body and are located after the </head> tag. Elements can contain attributes, these attributes have the meaning of a property of the element and serve for its visual design.

Headlines

A HTML heading or HTML h tag can be defined as a title or a subtitle which you want to display on the webpage. ... h1 is the largest heading tag and h6 is the smallest one.

Paragraph

The p HTML element represents a paragraph. Paragraphs are represented as blocks of text separated from adjacent blocks by blank lines and/or first-line indentation.

Reformatted text

The text enclosed by the pre tags is displayed in the document field without changing the fixed-width font. The command is used whenever the text needs to look the same as it was set in the source.

Comments

The text that is marked as a comment and it will not be interpreted and accordingly displayed in the field of the document.
